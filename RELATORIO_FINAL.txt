🔄 FLUXO DE DADOS
═══════════════════════════════════════════════════════════════════════════════

CRIAR:
   Clica + → Formulário → Preenche → Transpõe → Salva → localStorage → Lista

EDITAR:
   Clica card → Detalhes → Editar → Formulário preenchido + Parser → Modifica
   → Salva → localStorage → Lista atualizada

DELETAR:
   Clica card → Detalhes → Deletar → Confirmação → Remove → localStorage
   → Lista atualizada
